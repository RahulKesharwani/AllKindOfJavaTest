## About

this is a rest service containing, small example on [google-protobuf]().

- To make build this app, please run:
    
    
    mvn clean install
  
- To Run this app:

    
    mvn spring-boot:run
    
- Endpoints available:
```
    http://localhost:8080/courses/1
    
    http://localhost:8080/courses/2
```
